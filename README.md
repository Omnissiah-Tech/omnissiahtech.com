# omnissiahtech.com

Landing page for OmnissiahTech, parent company of [OmniGrade](https://omnigrade.ai).

Built with [Jekyll](https://jekyllrb.com/), served by GitHub Pages.

## Local development

Ruby version is pinned via `.ruby-version` (3.3.6). Use [rbenv](https://github.com/rbenv/rbenv) or another version manager to match.

    make install   # bundle install (first time only)
    make serve     # preview at http://localhost:4000

`make help` lists all targets.

## Deploy

Push to `main`. GitHub Pages builds from the repo root.
